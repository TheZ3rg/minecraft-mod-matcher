# 🔄 ModMatcher

**ModMatcher** — это удобное десктопное приложение для Windows, предназначенное для автоматического поиска и загрузки обновлений для модов Minecraft. 

Приложение находит моды из выбранной вами директории, находит их в огромной базе данных **Modrinth** и позволяет в один клик скачать актуальные версии модов под нужный загрузчик и версию игры.

![ModMatcher Icon](src/resources/icons/icon.png)
## ✨ Ключевые возможности

* 🔍 **Умное сканирование:** Распознавание модов по SHA-1 хэшу через API Modrinth. Если мода нет в базе, программа локально прочитает его метаданные (`fabric.mod.json`, `mods.toml`, `quilt.mod.json` и т.д.).
* ⚙️ **Поддержка всех популярных загрузчиков:** Fabric, Forge, NeoForge, Quilt и Legacy Forge.
* 💾 **Система бэкапов:** Создание ZIP-архива вашей текущей папки с модами в один клик, чтобы ничего не потерять при обновлении.
* ⚡ **Многопоточность:** Асинхронное многопоточное скачивание файлов, загрузка иконок и проверка обновлений — скорость это приоритет.
* 🎯 **Гибкая загрузка:** Возможность массового обновления всех модов или точечной загрузки только выделенных элементов.
* 📦 **Портативность:** Приложение поставляется в виде одного `.exe` файла и не требует установки дополнительных компонетов на утройство пользователя.

## 🚀 Установка и запуск (Для пользователей)

1. Перейдите на страницу [Releases](../../releases) в этом репозитории.
2. Скачайте последний архив `ModMatcher-v1.0.0-Windows.zip`.
3. Распакуйте архив в любую удобную папку.
4. Запустите `ModMatcher.exe`.

> **⚠️ Примечание:** Так как программа собрана в один исполняемый файл без платной цифровой подписи, Windows Defender или SmartScreen иногда могут выдать предупреждение при первом запуске. Для работы приложения нажмите *"Подробнее"* -> *"Выполнить в любом случае"*.

## 📖 Как пользоваться

1. **Укажите пути:** В верхней части окна выберите исходную папку с вашими старыми модами и папку, куда программа должна сохранить новые версии.
2. **Дождитесь сканирования:** Программа автоматически найдет все `.jar` файлы, высчитает их хэши и подгрузит красивые иконки.
3. **Выберите фильтры:** В правой панели выберите нужную версию Minecraft (например, `1.20.1`) и загрузчик (например, `Fabric`).
4. **Проверка:** Нажмите кнопку `Проверить наличие версий`. Программа свяжется с серверами Modrinth и покажет моды, для которых есть нужная вам версия.
5. **Загрузка:** Выберите моды (или оставьте все) и нажмите `Загрузить`. Новые `.jar` файлы появятся в выбранной вами папке сохранения!
